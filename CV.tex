\begin{itemize}[nosep, topsep=\intraProjectListTopSep]
    \item \textbf{架构分析与异步编排}：深入研读 Dify 的 \textbf{Flask + Celery} 分离式架构，理解其如何通过异步任务队列处理耗时操作（如文档解析、向量化），为后续 Web Agent 的异步工具调用提供了设计参考。
    \item \textbf{Agent + RAG 核心流程实践}：基于 Dify 的 \textbf{Agent} 与 \textbf{RAG} 模块，自主设计了一个企业知识库问答系统，实现了多轮对话记忆、文档溯源与检索增强生成，其中 Agent 能够根据用户问题动态选择检索策略。
    \item \textbf{工程化部署与落地}：使用 \textbf{Docker Compose} 一键部署 PostgreSQL、Redis、Weaviate 等中间件，将系统完整交付至生产环境，支撑了数千份内部文档的实时问答，平均检索响应在 1.5 秒以内。
\end{itemize}

    \item \textbf{深化高并发与工程能力}：借助 Go 语言的优势，对 Agent 的编排层进行了并发优化，使系统能同时处理数十路问答请求。最终沉淀出一套可复用的后端 Agent 服务模板，能为团队后续的 Web Agent 项目（如浏览器自动化 Agent、多 Agent 协作）提供了基础框架。
