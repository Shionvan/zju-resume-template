\textbf{基于 Dify 的 RAG 智能问答系统后端设计与部署} \hfill  \titlebreak
项目描述：基于 Dify 后端架构（Flask + Celery + DDD），设计并部署了企业级 RAG 智能问答系统，支持文档解析、向量检索与多轮对话。
% 在 itemize 的选项中，使用 topsep=\intraProjectListTopSep 来控制上边距
\begin{itemize}[nosep, topsep=\intraProjectListTopSep]
    \item \textbf{系统架构分析}：深入研读 \textbf{Dify} 的后端架构设计，理解其基于 \textbf{Flask + Celery} 的API服务与异步任务分离架构，以及\textbf{领域驱动设计（DDD）}的分层理念。
    \item \textbf{核心流程实践}：基于 Dify 的 \textbf{Agent + RAG} 模块，设计并实现了一个企业内部知识库问答系统，支持多轮对话与文档溯源。
    \item \textbf{工程化部署}：使用 \textbf{Docker Compose} 完成 PostgreSQL、Redis、Weaviate 等中间件的一键部署，让系统能落地到生产环境。
\end{itemize}
